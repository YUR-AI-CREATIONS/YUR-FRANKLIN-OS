# Where is the Code? 🔍

A powerful Python utility to locate, analyze, and organize code files across your projects.

## Features

- 🔍 **Smart Code Detection**: Automatically identifies code files by extension
- 🚀 **Project Recognition**: Detects different project types (Python, Node.js, Rust, etc.)
- 📊 **Statistics**: Provides detailed analysis of your codebase
- 🎯 **Pattern Matching**: Search for files using wildcards
- 📁 **Directory Traversal**: Intelligently skips common non-code directories
- 🎨 **Multiple Output Formats**: Table or JSON output
