FALLBACK ROUTING STRATEGY
============================================================

If Primary Tier Fails:

Tier 1 (Cognitive Engine) Fails:
  → Try Tier 2 (Bootloader reasoning)
  → Fallback: Yur cached response

Tier 2 (Bootloader) Fails:
  → Try Tier 1 (Cognitive Engine analysis)
  → Fallback: Yur communication + retry later

Tier 3 (Yur) Fails:
  → Try Tier 1 (Cognitive Engine) for communication text
  → Try Tier 2 (Bootloader) for execution

All Tiers Fail:
  → Return 503 Service Unavailable
  → Check Ollama (localhost:11434)
  → Check Yur (localhost:3000)
  → Check Cognitive Engine path (F:\)
